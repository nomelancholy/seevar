# 추후 업데이트 고려 사항

사용자·커뮤니티 제안을 정리해 두고, 우선순위·리소스에 따라 반영을 검토하기 위한 문서입니다.

---

## 링크·공유·딥링크

### 이의 제기 등 열려 있는 컨텍스트로 바로 열리는 링크

**제안 요지**  
이의 제기(모멘트 댓글 모달)처럼 현재 열려 있는 컨텍스트에 대한 **공유 가능한 링크**가 있으면, 링크를 받은 사람이 같은 화면(해당 모멘트·댓글 모달 등)으로 바로 진입할 수 있어서 좋겠다는 의견.

**고려할 점**

- 경기 상세에서 **`openMoment`** 쿼리는 이미 사용 중 (`?openMoment=<momentId>` → 해당 쟁점 순간 카드 클릭 후 모달 열림). 이 패턴을 확장해 **모달이 열린 상태의 URL**을 공유하면, 링크만으로 같은 컨텍스트(해당 모멘트·댓글 화면)로 진입할 수 있게 할 수 있음.
- 예: `?openMoment=<momentId>` 진입 시 모달 자동 오픈은 이미 동작. 추가로 `?openComment=<commentId>` 등으로 특정 댓글/스레드로 스크롤·포커스까지 지정할지 검토.
- 공유 시 **현재 URL(쿼리 포함)**을 복사해도 동일 컨텍스트로 열리도록 유지·보완.

**관련 코드·기능**

- 경기 상세: `searchParams.openMoment`, `MatchMomentCards`의 `initialOpenMomentId`.
- 댓글/알림 링크: `lib/actions/comments.ts`, `admin-reports.ts`에서 `openMoment` 쿼리 생성.
- 모멘트 댓글 모달: `MomentCommentModal`, `/api/moments/[id]`.
